Markdown ​

Markdown 是一種輕量級標記式語言，它有純文字標記的特性，讓編寫的可讀性提升，並且Markdown可以混合 HTML 一同使用。 Markdown同時擁有了HTML的特性同時也比HTML還要易於撰寫。
